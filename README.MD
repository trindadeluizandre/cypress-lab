Cypress Automation Laboratory

This repository serves as a technical environment for Software Quality Assurance (QA) and Automated Testing development. It is structured to demonstrate E2E (End-to-End) testing capabilities using the Cypress framework.

Project Scope
The objective of this project is to implement and maintain a suite of automated tests for web applications. The repository focuses on functional validation, regression testing, and the application of industry-standard QA methodologies.

Technical Stack
Automation Framework: Cypress 13.0.0 or higher.
Programming Language: JavaScript.
Runtime Environment: Node.js.
Development Environment: Visual Studio Code.
Version Control: Git / GitHub.

Test Case Documentation
Suite: Authentication and User Access.
CT003: Validation of successful authentication and URL parameter persistence.
CT004: Validation of mandatory field constraints for the password input.

Execution Procedures
Prerequisites: Ensure that the Node.js environment is correctly configured on the local machine.
Installation:

git clone https://github.com/trindadeluizandre/cypress-lab.git

npm install
Test Execution:
Interactive Mode (Cypress Runner): npx cypress open
Headless Mode (CLI): npx cypress run

Laboratório de Automação Cypress

Este repositório serve como um ambiente técnico para garantia de qualidade de software (QA) e desenvolvimento de testes automatizados. Está estruturado para demonstrar capacidades de testes E2E (ponta a ponta) utilizando o framework Cypress.

Escopo do Projeto
O objetivo deste projeto é implementar e manter uma suíte de testes automatizados para aplicações web. O repositório foca em validação funcional, testes de regressão e aplicação de metodologias de QA padrão da indústria.

Stack Técnica
Framework de Automação: Cypress 13.0.0 ou superior.
Linguagem de Programação: JavaScript.
Ambiente de Execução: Node.js.
Ambiente de Desenvolvimento: Visual Studio Code.
Controle de Versão: Git / GitHub.

Documentação de Casos de Teste
Suíte: Autenticação e Acesso de Usuário.
CT003: Validação de autenticação bem-sucedida e persistência de parâmetros de URL.
CT004: Validação de restrições de campos obrigatórios para o campo de senha.

Procedimentos de Execução
Pré-requisitos: Certifique-se de que o ambiente Node.js está configurado corretamente na máquina local.
Instalação:

git clone https://github.com/trindadeluizandre/cypress-lab.git

npm install
Execução de Testes:
Modo Interativo (Cypress Runner): npx cypress open
Modo Headless (CLI): npx cypress run
